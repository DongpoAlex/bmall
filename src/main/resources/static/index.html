关于我们

陕西米禾供应链管理股份有限公司成立于2015年，由十余家食材行业领军企业组建而成。 公司集食材批发、加工、分拣、分装、配送、冷链仓储、物流服务、电商交易平台、连锁超市、产业配套于一体。 主要从事“生鲜食材+互联网平台+校企、部队餐厅/厨房食材/社区超市”的业务运营，包括线下传统业务、线上展示交易、全程冷链配送， B2B/B2C等运营模式。以实体销售+互联网营销的方式，实现农产品质量源头追溯，全程冷链运输监控， 确保食品安全。优化食材供应链管理，实现互联网和物联网的有机结合。打造国内一流的农产品流通产业链的综合交易运营服务平台。
